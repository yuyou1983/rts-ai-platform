# 回放分析

此技能深度分析对局回放，提取可执行洞察。

## 分析维度

### 1. 事件提取
- 生产事件：单位生产、建筑建造、科技升级
- 战斗事件：交火、战损、技能使用
- 经济事件：采集、扩张、资源转换
- 侦察事件：视野获取、情报更新

### 2. 时间线分析
```
Timeline:
0:00 ──────────────────────────────────────────── 15:00
  │                                                    │
  │  ★ 扩张    ★ 战斗    ★ 科技     ★ 总攻            │
  │  5:30      8:45     12:00     14:30              │
  │                                                    │
  └────────────────────────────────────────────────────┘
```

### 3. 关键指标计算
- 有效 DPS：实际输出/理论最大输出
- 资源利用率：采集效率/饱和度
- APM：每分钟操作数
- 决策延迟：状态观察到命令发出的时间

### 4. 失误识别
- 错过时机：扩张/科技/进攻时机延迟
- 微操失误：单位死亡可避免
- 宏观失误：战略方向错误
- 资源浪费：资源积压或错误分配
