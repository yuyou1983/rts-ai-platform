01 / VISION

AI Native RTS：不是给 RTS 加 AI，
而是让 AI 驱动 RTS

传统 RTS

引擎 → 规则 → 美术 → 关卡 → 最后塞一个 AI
AI 是事后补丁，受限于引擎 API、单线程、不可训练

AI Native RTS

协议优先 → 可训练仿真 → 多 Agent 编排 → 引擎无关渲染
AI 是一等公民，可训练、可回放、可批量仿真、可进化
ADR-1
渐进式 Agent 拆分
M0 单体 → M1 三核心 → M2 全 Agent
ADR-2
协议隔离 (Protocol Barrier)
Agent 只见 obs/cmd，不耦合引擎
ADR-3
LLM 不进高频环
线上用蒸馏模型，LLM 仅离线规划
